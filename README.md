## 動作環境

本プロジェクトは以下の環境で開発・動作確認を行っています。

### 開発・動作確認済み環境
| 項目 | 内容 |
| :--- | :--- |
| **OS** | Windows 11 Pro 64bit (Version 25H2) |
| **CPU** | Intel Core i7-13700F (2.10 GHz) |
| **GPU** | NVIDIA GeForce RTX 4060 Ti (8GB) |
| **RAM** | 32GB |
| **DirectX** | Version 11 (DxLib使用) |
| **ストレージ** | 空き容量 1GB 以上推奨 |

### 開発ツール
* **言語:** C++
* **ライブラリ:** DxLib
* **IDE:** Visual Studio 2022

## 導入手順

1. 本リポジトリ右側の **[Releases]** ページへアクセスします。
2. 最新のZIPファイルをダウンロードし、展開します。
3. フォルダ内の `3DShooting.exe` を実行してください。
   
   ※セキュリティ警告が出る場合がありますが、そのまま実行を許可してください。
